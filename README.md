# 4Chan-Da-Sala

O 4Chan-Da-Sala é um aplicativo mobile que simula um fórum anônimo voltado para os alunos da nossa sala. A proposta é permitir que os usuários compartilhem sentimentos, opiniões e piadas de forma livre, mantendo sempre o anonimato com segurança.

## Objetivo

Criar um ambiente onde todos possam se expressar sem medo de julgamento, incentivando a interação e o compartilhamento dentro da sala.

## Regras e Moderação

Apesar de ser um fórum anônimo, o app conta com moderação ativa:

* Conteúdos ilícitos não são permitidos
* Linguagem ofensiva ou imprópria resultará em banimento
* A identidade dos usuários será mantida em segredo
* A moderação garante um ambiente saudável para todos

## Funcionalidade Atual

Atualmente, o projeto consiste em uma tela de login, com as seguintes funcionalidades:

* Alternância entre login por email ou número
* Campo de senha com ocultação de texto
* Botão de recuperação de senha
* Botão de cadastro (Sign Up)
* Interface estilizada com NativeWind

## Tecnologias Utilizadas

* React Native: desenvolvimento mobile
* Expo: ambiente de desenvolvimento simplificado
* TypeScript: tipagem estática
* NativeWind: estilização com Tailwind no React Native

## Estrutura do Projeto

```
components/           # Componentes reutilizáveis (ex: inputs)
assets/               # Imagens e recursos visuais
styles/               # Estilos globais
App.tsx               # Tela principal (login)
```

## Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm run start
```

3. Escolha a plataforma no Expo:

```bash
i  # iOS
a  # Android
w  # Web (quando disponível)
```
